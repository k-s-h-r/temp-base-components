## 個別間隔指定

子要素それぞれに間隔を指定することができます。  
Stackコンポーネント直下の要素のclassに `!only-stack-space-[*]` を指定することで、その要素のみに間隔を指定することができます。  
`!only-stack-space-[*]` には `px`, `0`, `2`, `4`, `6`, `8`, `10`, `12`, `14`, `s`, `m`, `l`, `xl` を指定することができます。

</Canvas>
