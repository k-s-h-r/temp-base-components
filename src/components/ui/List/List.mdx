## 数字リスト
[list-style-type](https://developer.mozilla.org/ja/docs/Web/CSS/list-style-type)を指定することで、数字リストのビュレットを任意に変更にすることができます。

例：
- `list-[lower-roman]`
- `list-[cjk-earthly-branch]`
- `list-[hiragana-iroha]`

<Canvas>
  <Story of={Stories.OrderredOtherListStyleType} />
</Canvas>


## 数字リスト - 逆順

論理属性で、リストの項目が逆順で指定されていることを指定します。  
項目は大きい方から小さい方へ番号付けされます。

<Canvas>
  <Story of={Stories.OrderredReverse} />
</Canvas>


## 数字リスト - 開始値

整数値を持つ属性で、リスト項目の序数の開始値を指定します。  
この値は、番号付けの種類が文字やローマ数字であっても、常にアラビア数字 (1, 2, 3, など) で指定します。  
例えば、 "d" の文字や "iv" のローマ数字から始める場合は、 start="4" を使用してください。

</Canvas>
