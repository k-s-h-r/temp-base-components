## External

外部リンクの場合、指定します。

  <Story of={Stories.ExternalLink} />

## No Link

`isLink=false`、もしくは`href`指定なしでリンクを無効にすることができます。  
`isDisabled`との違いはスタイルで、`isLink=false`の場合、薄くならずデフォルトテキスト色になります。

<Canvas>
  <Story of={Stories.NoLink} />
</Canvas>

## No Line

下線を使用したくない場合、`isLine=false`を指定します。

  <Story of={Stories.NoLine} />
</Canvas>

## Disabled

disabledにする場合、指定します。  
hrefの指定は不要です。  
押したりフォーカスを受けたりすることができなくなります。

  <Story of={Stories.DisabledLink} />
